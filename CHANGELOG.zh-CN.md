# 版本记录

本文件记录项目的重要变更。

## [1.0.2] - 2026-07-17

### 新增

- 新增贡献指南、数据来源、安全策略、协作行为规范和版本记录的中文版本。
- README 增加中英文文档入口和相互链接。

## [1.0.1] - 2026-07-17

### 修改

- GitHub Actions 新增离线 GraphRAG 图谱检索冒烟测试。
- 发布校验新增安全 LLM 配置模板和关键 Git 忽略规则检查。
- 新增协作行为规范和仓库检索标签。

## [1.0.0] - 2026-07-17

### 新增

- 发布以结直肠癌 NCIt 根概念 `C2955` 为中心的研究型知识图谱。
- 提供可复现的 NCIt 抽取、关联实体补全、图谱构建、增量更新、可视化和本地问答流程。
- 增加公开发布文档：数据来源、引用元数据、贡献指南、安全策略和自动验证。
- 提供本地工作台，用于图谱浏览、新资料抽取、人工核查合并和 GraphRAG 问答。

### 说明

- 图谱用于术语整理和研究支持，不是临床决策支持系统，也不能替代个体化诊疗建议。
- API 密钥、本地运行结果和可选 OCR 资源不纳入版本控制。
