# Changelog

[中文版](CHANGELOG.zh-CN.md)

All notable changes to this project are documented in this file.

## [1.0.2] - 2026-07-17

### Added

- Chinese counterparts for contribution, data-provenance, security, conduct, and changelog documentation.
- Cross-links between English and Chinese documentation from the main README.

## [1.0.1] - 2026-07-17

### Changed

- Added an offline GraphRAG retrieval smoke test to GitHub Actions.
- Extended release validation to enforce safe LLM example configuration and critical Git ignore rules.
- Added a contributor code of conduct and repository discovery topics.

## [1.0.0] - 2026-07-17

### Added

- Public research release for the colorectal cancer NCIt knowledge graph rooted at `C2955`.
- Reproducible NCIt extraction, related-entity enrichment, graph construction, incremental update, visualization, and local question-answering workflow.
- Publication documentation: README, data provenance, citation metadata, contribution guidance, security policy, and automated release validation.
- Local graphical workbench for graph browsing, incremental material extraction, review-oriented merging, and GraphRAG-style question answering.

### Notes

- The graph is a terminology and research-support resource. It is not a clinical decision-support system and must not be used as individualized medical advice.
- API keys, local runtime outputs, and optional OCR resources are excluded from version control.
