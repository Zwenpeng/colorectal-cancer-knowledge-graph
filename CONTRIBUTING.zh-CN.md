# 贡献指南

提交修改时，应保持 NCIt 标识符、原始关系方向、来源可追溯性，以及“已有节点”和“未审核候选实体”的边界。

提交前请确认：

1. 不提交 API 密钥、患者资料或受限文档。
2. NCIt 编码必须稳定保留，不能用显示名称替代节点标识符。
3. 原始图谱输出中不得随意改变关系方向。
4. 增量候选节点或关系必须附带证据与来源元数据。
5. 发起 Pull Request 前运行 `python .\scripts\validate_release.py` 和 Python 语法检查。

更新术语数据时，请说明 NCIt 版本、抽取日期、根节点编码，以及预期新增或变化的节点数、边数。
